php多线程

发表于 2018-04-06

昨天被一个同事问道：有没有用过php多线程？

两眼一黑，答：没用过。

后来查资料得知，php其实也是支持多线程，要使用是需要安装pthread扩展。

在 (http://php.net/manual/zh/intro.pthreads.php) 中写着：

不可以在 web 服务器环境中使用 pthreads 扩展， PHP 多线程开发仅限于命令行模式的应用。

只能在 PHP 7.2+ 版本中使用 pthreads 扩展， 因为在之前的 PHP 版本中，ZTS 模式是不安全的。
